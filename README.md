# CheckIpUpdate
Check the IP address of your ddns domain name and write the LeCDN forwarding IP address | 检查你的ddns域名的ip并写入LeCDN转发ip

主要用于解决LeCDN的四层转发的源站ip无法填域名的问题（某些家里云可能需要填入使用了ddns的域名）
